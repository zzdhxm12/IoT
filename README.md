# IoT
 Smart package locker
 스마트 택배 보관함
 
외출이 잦은 사람들은 택배를 배송 시킬 경우 택배를 무사히 받을 때 까지 도난의 위험에 불안하기 때문에 귀가 전까지 택배를 안전하게 보관할 수 있는 스마트 택배 보관함을 개발
